--------------------------------------------------------
  FUNCIONALIDADES DESTACADAS DEL SISTEMA
--------------------------------------------------------

  1. AUTENTICACIÓN SEGURA
     - Login con protección brute-force (bloqueo tras 5
       intentos fallidos en 15 minutos)
     - Sesión con timeout automático de 30 minutos
     - Tokens CSRF en todos los formularios
     - Cookies con SameSite=Strict y HttpOnly

  2. ROLES DIFERENCIADOS
     - Administrador: control total del sistema
     - Profesor: gestión de sus cursos y calificaciones
     - Alumno: vista de su expediente académico

  3. SEGURIDAD
     - Prepared Statements en todas las consultas SQL
       (protección contra SQL Injection)
     - Cabeceras HTTP de seguridad (X-Frame-Options,
       Content-Security-Policy, HSTS en HTTPS)
     - Log de auditoría: toda acción queda registrada
       con usuario, fecha, tabla e IP

  4. DISEÑO RESPONSIVE
     - Funciona en escritorio, tablet y móvil
     - Modo oscuro disponible (botón ☀️/🌙 en la barra)

  5. OTRAS FUNCIONES
     - Mensajería interna entre usuarios
     - Exportación a PDF y Excel de calificaciones
     - Avisos/anuncios del sistema
     - Subida de foto de perfil
     - Notificaciones en tiempo real (polling)

--------------------------------------------------------
  TECNOLOGÍA UTILIZADA
--------------------------------------------------------

  Backend  : PHP 8.x (sin frameworks)
  Base de datos: MySQL 8 (Railway cloud)
  Frontend : HTML5 + CSS3 + JavaScript vanilla
  Despliegue: Railway (CI/CD automático via GitHub)
  Servidor : Caddy (en contenedor Docker)
  Seguridad: CSRF, rate limiting, prepared statements,
             cabeceras HTTP, SameSite cookies, HSTS
